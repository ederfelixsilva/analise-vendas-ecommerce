# 📊 Análise de Vendas de E-commerce

Projeto desenvolvido para praticar Análise de Dados com Python e Pandas.

## Tecnologias

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objetivos

- Explorar um conjunto de dados de vendas
- Realizar limpeza e preparação dos dados
- Gerar insights de negócio
- Criar visualizações

## Estrutura

- data/: conjunto de dados
- notebooks/: análises
- scripts/: geração do dataset
